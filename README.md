# ⚡ Postman-Lite

A lightweight, modern API testing client inspired by Postman, built with **React** and **Vite**. It provides a clean interface for sending HTTP requests, managing headers, query parameters, request bodies, and viewing formatted API responses.

## 🌐 Live Demo

🔗 https://postman-lite-chi.vercel.app/

---

## ✨ Features

- 🚀 Send HTTP requests
  - GET
  - POST
  - PUT
  - PATCH
  - DELETE

- 📋 Custom request headers
- 🔍 Query parameter support
- 📦 JSON request body
- 📄 Formatted JSON response viewer
- 📊 Response status
- ⚡ Response time
- 📐 Response size
- 🎨 Clean and responsive UI
- 🌙 Modern developer-friendly interface

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/abrarhussain01/Postman-Lite.git
```

### Navigate into the project

```bash
cd Postman-Lite
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
